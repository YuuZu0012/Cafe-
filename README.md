วิธีการใช้เว็ป
1. เราสามารถล็อคอินเข้าไปในเว็ปของเราได้
2. เริ่มแรกเราจะเป็นUser เราสามารถที่จะสั่งซื้อสินค้าได้ตามเว็ปจะมีบอกในส่วนของราคา ชื่อ และคำอธิบาย และเมื่อเราสั่งซื้อจะพาไปหน้าถัดไป
3.เมื่อเราได้ของที่อยากได้และกดใส่ไปรายการจะมีการคำนวฯอัตโนมัติและจะมีในส่วนของโปรโมชั่นและส่วนลด เราสามารถเลือกได้ตามที่ต้องการง
4.เมื่อเรากดยืนยันการสั่งซื้อเว็ปจะพาเราไปที่ หน้าการสั่งซื้อโดยจะมีส่วนของQR กับจ่ายผ่านธนาคารและเมื่อเราเลือกได้แล้ว เว็ปจะถามเราอีกครั้งว่าต้องการสั่งซื้อจริงๆใช่มั้ย และเมื่อเราตอบตกลง เว็ปจะขึ้นยินยันการสั่งซื้อสำเร็จ

ส่วนของแอดมิน ต่อ
1.เมื่อ user ได้สั่งเข้ามาแล้วจะเด้งไปเข้าที่รายงานสินค้าว่าuser ได้สั่งอะไรไปรวมถึงยอดเงินสินค้า ที่หักจะส่วนลดแล้ว และข้อมูลการสั่งซื้อจะไปร่วมกับกราฟยอดขายอีกด้วยว่าได้เงินเท่าไหร่แล้ว  และสินค้าที่ขายดีที่สุดคืออะไร
2.และadminจะเข้าได้ถึงทุกส่วยของเว็ป และเช็ดยอดขายและแก้ข้อมูลของuser และพนักงานภายในร้านได้ และเพิ่มเมนูอาหารใหม้

ส่วนพนักงาน
1.พนักงานจะสามารถดูรายการอาหารได้และร่วมถึงการสั่งซื้อของลูกค้า 
2.พนักงานจะไม่มีสิทธื์มากเท่าadmin ที่สามารถเข้าได้ทุกอย่าง พนักงานจะสามารถเช็คข้อมูลบางส่วนขั้นต้นได้ แต่เมื่อจะเช็ดระดับสูง ทางเว็ปจะมีการแจ้งเตือน หรือไม่ขึ้นเลย ในส่วยของหลังบ้านที่มีแต่admin
